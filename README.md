# RTK Control Center - Versión corregida

## Ejecutar

```bash
cd RTK_Control_Center_CORREGIDO
python3 -m pip install -r requirements.txt
streamlit run app.py
```

## Datos
La base local está en:

```text
data/rtk_local.csv
```

Para actualizarla, reemplaza ese archivo por el CSV nuevo manteniendo el mismo nombre.
